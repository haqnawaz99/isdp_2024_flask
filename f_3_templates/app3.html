Using For Loops in Flask Templates

Introduction to Templates in Flask

Flask allows you to use templates to render HTML dynamically based on the data provided by your Python code. Templates in Flask are powered by the Jinja2 template engine, which provides features such as template inheritance, conditional statements, loops, and more.

Passing Data from Flask to Templates

In Flask, you can pass data from your routes (Python functions) to the HTML templates. This is done using the render_template() function, which takes the name of the template file and any data you want to pass as keyword arguments.

For example, you can pass an array from your Python code to an HTML template and then use a loop in the template to display each item in that array.

Example Code

Python Code (app.py)

Hereâ€™s an example of how to pass an array of fruits from your Flask application to an HTML template:

from flask import Flask, render_template

app = Flask(__name__, template_folder='templates')

@app.route('/')
def index():
    myarray = ['Apple', 'Banana', 'Cherry', 'Date', 'Elderberry']
    return render_template('index.html', fruits=myarray)

if __name__ == '__main__':
    app.run(host='0.0.0.0', debug=True)

In this code:

  • myarray is a list of fruit names.
  • This list is passed to the template index.html with the keyword fruits.

HTML Template (index.html)

In your HTML template, you can loop through the array and display each item using a for loop in Jinja2 syntax:

Understanding the Code

In the HTML template:

  • The {% for fruit in fruits %} syntax is a Jinja2 loop that iterates over each item in the fruits array passed from the Flask application.
  • For each iteration, the current item (fruit name) is displayed within a <li> element.

Running the Application

To run the application:

  • Save the Python code in a file named app.py.
  • Create a directory named templates and save the HTML code in a file named index.html inside this directory.
  • Run the Flask application using the command: python app.py.
  • Open your web browser and go to http://localhost:5000 to see the dynamically generated list of fruits.

Conclusion

This example shows how to pass data from your Flask application to an HTML template and use a for loop in the template to render dynamic content. This is a powerful feature of Flask that allows you to create dynamic, data-driven web pages with ease.
